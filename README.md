## 安装方式
### ⚠️ 浏览器安装须知
由于Edge/Chrome安全策略，**直接下载CRX文件会被浏览器自动删除、即使安装后也无法启用**。
仓库内保留CRX文件仅作历史存档备份，不再推荐CRX直装方式。若需下载CRX，需右键下载链接，将其另存为zip，以zip格式下载再更改CRX格式。

✅ 可用安装方式：下载源码压缩包，浏览器开启开发者模式，使用「加载已解压的扩展程序」安装即可正常使用。
1. 浏览器打开扩展管理，开启「开发者模式」
2. 下载本项目的源码文件，拖入扩展页面安装
