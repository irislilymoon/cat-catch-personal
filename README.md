# Cat Catch Personal Build
基于原版猫抓的个人构建版本

## 项目说明
本项目基于 **猫抓 (Cat Catch)** 源码进行个人定制修改，仅用于个人学习与技术交流。
原项目地址：https://github.com/xifangczy/cat-catch
原协议：**GNU General Public License v3.0**

## 说明
本版本为个人自用定制，仅做了少量非功能性调整，核心功能与原版保持一致。

## 安装方式
1. 浏览器打开扩展管理，开启「开发者模式」
2. 下载本项目的 `.crx` 文件，拖入扩展页面安装
3. 或直接「加载已解压的扩展程序」使用源码
